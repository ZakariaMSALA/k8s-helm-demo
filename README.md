## 📦 Project Overview
This repository demonstrates deploying a simple app with Helm on Kubernetes.

**Stack:**
- Kubernetes
- Helm

## 🚀 How to Deploy
```bash
cd helm
helm install myapp ./myapp
kubectl get all
helm uninstall myapp
```

## 🔧 Customization
Edit `values.yaml` to change image, replicas, or ports.
